# Token Hunter Foundation Pack

Tento balík je výchozí source-of-truth pro projekt **Token Hunter**.

Obsah:
- `PROJECT_COMPASS.md` — identita produktu, principy, anti-goals, cílovka, JTBD
- `SCOPE.md` — co je v produktu, co není, scope guardrails
- `PRODUCT_MAP.md` — detailní mapa funkcí, domény, obrazovek a toků
- `ROADMAP.md` — pořadí výstavby produktu po etapách
- `DECISIONS.md` — závazná architektonická a produktová rozhodnutí
- `SOURCE_POLICY.md` — pravidla práce se zdroji a ověřováním
- `SCORING_POLICY.md` — jak se počítá best-buy, confidence a game-changer
- `IOS_STRATEGY.md` — iOS směr, omezení platformy, MVP a V2
- `MVP_PLAN.md` — konkrétní realistický MVP plán
- `REFERENCES.md` — externí zdroje a inspirace

Zlatá věta projektu:

> Token Hunter není katalog modelů. Je to rozhodovací motor, který řekne, jak za AI utratit co nejmíň a přitom nekoupit blbost.
